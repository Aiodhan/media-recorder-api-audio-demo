Recording audio using MediaRecorder API

The MediaRecorder API makes media recording in the browser super easy. It allows storing chunks of data from a media stream as blobs, which can later be concatenated as a single file. The file can then be saved locally or uploaded/POSTed to the web server.

Unfortunately it is only supported by Chrome, Firefox and Opera. Please comment on the Safari/WebKit feature request and the Edge feature request.
